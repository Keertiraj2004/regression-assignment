# Regression Assignment using Machine Learning

## 📌 Project Overview
This project demonstrates the implementation of different **Regression Algorithms** in Machine Learning using Python and JupyterLab.

The assignment covers:

- **Simple Linear Regression**
- **Multiple Linear Regression**
- **Polynomial Regression**
- **Ridge Regression**
- **Lasso Regression**

The goal is to understand how different regression models work and compare their performance using evaluation metrics.

---

## 🎯 Objective
The objective of this assignment is to:

- Learn how regression algorithms are applied to different datasets
- Understand the difference between linear and nonlinear regression
- Study the impact of regularization techniques such as Ridge and Lasso
- Compare model performance using metrics like **R² Score** and **Mean Squared Error (MSE)**
